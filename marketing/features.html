Preguntas Frecuentes

¿Necesita internet para funcionar?

El sistema funciona al 100% en red local (LAN). Solo se requiere internet para validar la licencia una vez cada 30 días.

¿Es compatible con volantes Direct Drive?

Sí, detecta y configura cualquier dispositivo DirectX: Fanatec, Simucube, Moza, Thrustmaster y Logitech.

¿Puedo usar mis propios mods?

Absolutamente. El sistema es abierto. Puedes subir mods de RaceDepartment o privados.
