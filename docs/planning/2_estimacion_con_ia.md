# Estimación con Asistencia de IA (Agentic Coding)

El uso de asistentes de IA avanzados (como el que estás usando ahora) reduce drásticamente los tiempos de **escritura de código (boilerplate)**, **refactorización** y **generación de tests**.

La reducción estimada general es del **40% al 50%**, aunque la **supervisión humana** y las **pruebas físicas** (con los simuladores reales) siguen requiriendo tiempo fijo.

## Nueva Estimación de Tiempo

| Fase | Estimación Manual | Estimación con IA | Ahorro |
| :--- | :--- | :--- | :--- |
| **Backend** | 120 - 160h | **60 - 80h** | ~50% |
| **Frontend** | 140 - 180h | **70 - 90h** | ~50% |
| **Agente** | 80 - 120h | **50 - 70h** | ~35% |
| **DevOps/QA** | 40 - 60h | **25 - 35h** | ~40% |
| **TOTAL** | **380 - 520h** | **205 - 275h** | **~45%** |

## Nuevo Plazo de Entrega
*   **1 Desarrollador + IA:** 1.5 - 2 meses (vs 3-4 meses).

## Nuevo Coste Estimado
Al reducir las horas, el coste total del proyecto disminuye proporcionalmente.

*   **Opción Freelance (25€-40€/h):** 5.100 € - 11.000 €
*   **Opción Senior/Agencia (50€-80€/h):** 10.250 € - 22.000 €

## ¿Dónde ayuda más la IA?
1.  **Backend:** Generar APIs CRUD, esquemas de base de datos y scripts de migración es casi instantáneo.
2.  **Frontend:** Crear componentes React y estilarlos con Tailwind es extremadamente rápido.
3.  **Tests:** La IA puede escribir el 80% de los tests unitarios automáticamente.

## ¿Dónde NO ahorra tanto tiempo?
1.  **Lógica de Negocio Compleja:** Las reglas específicas de tus torneos o la forma exacta de calcular el ELO requieren diseño y revisión humana.
2.  **Hardware:** Probar que el Agente funciona en el PC del simulador real, que se abre el juego, que detecta el volante... eso hay que probarlo manualmente.
